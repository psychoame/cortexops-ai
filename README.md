# CortexOps AI 🚀

Autonomous AI Incident Intelligence Platform for DevOps and SRE Teams.

## 📌 Problem

Engineering teams waste valuable time during outages manually analyzing logs, dashboards, and alerts to identify root causes.

Traditional monitoring tools detect symptoms — not causes.

---

## 💡 Solution

CortexOps AI is an AI-powered observability platform that:

- simulates infrastructure incidents
- streams live operational logs
- analyzes telemetry using AI
- identifies probable root causes
- recommends remediation actions
- visualizes incidents through a modern dashboard

---

## ✨ Features

- ⚡ Real-time incident simulation
- 📜 Live streaming log terminal
- 🧠 AI-powered root cause analysis
- 📊 Interactive infrastructure metrics
- 🚨 Severity classification system
- 🕒 Incident replay timeline
- 🎨 Premium enterprise dashboard UI
- 🌌 Cinematic observability experience

## 🚀 Running Locally

### Backend

```bash id="7jlwm8"
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000
```

### Frontend

```bash id="5jlwm6"
cd frontend
npm install
npm run dev
```

Frontend runs at:
```text id="9jlwm4"
http://localhost:3000
```
